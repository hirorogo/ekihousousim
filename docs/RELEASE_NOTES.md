# リリースノート

## バージョン 1.0.0 (2025年11月9日)

### 新機能

#### フロントエンドOCR機能
- tesseract.js を使用した日本語OCR機能を実装
- 画像ファイル（PNG, JPEG）からのテキスト抽出
- リアルタイム進捗表示
- OCR実行のオン/オフ選択機能

#### PDF変換・結合機能
- pdf-lib を使用した画像→PDF変換機能
- 複数PDFファイルの結合機能
- アスペクト比を考慮した画像配置
- 変換・結合の進捗表示

#### 複数ファイルアップロード機能
- 複数ファイル同時選択・アップロード
- ファイルプレビュー機能（名前、サイズ、形式表示）
- 個別ファイル削除機能
- 対応形式: PDF, JPEG, PNG, TXT

#### APIサーバー実装
- Node.js + Express を使用したAPIサーバー
- ファイルアップロードAPI（multer使用）
- JSONファイルベースの簡易データベース
- CORS対応によるクロスオリジン通信

#### 資料管理システム
- 資料の一覧表示・検索・フィルター機能
- リアルタイム検索とキーワードフィルタリング
- 科目別フィルター（6科目対応）
- 資料詳細ページ（基本実装）

#### WEBビューアー機能
- react-pdf を使用したPDFビューアー
- 画像ビューアー（PNG, JPEG対応）
- テキストファイルビューアー（プレーンテキスト）
- ページナビゲーション機能（PDF用）
- レスポンシブ表示対応

### 技術的改善

#### アーキテクチャ
- フロントエンド: React 19.1.1 + Vite 7.1.7
- バックエンド: Node.js + Express
- ルーティング: React Router DOM 7.9.5
- スタイリング: CSS Modules（グラデーション不使用）

#### API設計
- RESTful API設計
- 統一されたエラーハンドリング
- API_ENDPOINTSによる一元管理
- 環境変数対応（VITE_API_URL）

#### セキュリティ
- ファイル形式・サイズ制限（50MB）
- XSS対策（React標準）
- パストラバーサル対策
- CORS設定

### 対応ファイル形式
- **PDF**: application/pdf
- **画像**: image/jpeg, image/jpg, image/png  
- **テキスト**: text/plain
- **最大ファイルサイズ**: 50MB

## 既知の問題・制限事項

### 現在のバージョン（1.0.0）
- OCR機能は日本語のみ対応
- PDF結合は同一フォーマットファイルのみ推奨
- 認証システムは開発環境用（共通パスワード）
- ファイルストレージはローカルのみ
- データベースはJSONファイルベース

### 技術的制約
- tesseract.js の処理速度はファイルサイズに依存
- pdf-lib による変換はブラウザのメモリ制限あり
- 大容量ファイルの処理時間が長い場合がある
