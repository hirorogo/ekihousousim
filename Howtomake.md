# テスト過去問・プリント共有サイト 開発指導書

## はじめに
この指導書は、初心者の高校生がReact + Viteを使ってテスト過去問・プリント共有サイトを開発するためのステップバイステップガイドです。

## 開発環境の確認

### 必要なツール
- Node.js (v20.11.0以上)
- npm (Node.jsに含まれている)
- Visual Studio Code (推奨エディタ)
- Git

### 環境確認コマンド
```bash
# Node.jsのバージョン確認
node --version

# npmのバージョン確認
npm --version

# Gitのバージョン確認
git --version
```

## 第2段階: 基本的なページ構造の作成

### 優先順位の高いページから作成
1. **Homeページ** - サイトのトップページ
2. **Loginページ** - ユーザーログイン
3. **MaterialListページ** - 資料一覧

### 作成手順
1. 各`.jsx`ファイルに基本的なReact関数コンポーネントを作成
2. 対応する`.module.css`ファイルにスタイルを追加
3. App.jsxでルーティングを設定

### コンポーネント作成の基本パターン
```jsx
// 例: Home.jsx
import React from 'react';
import styles from '../css/Home.module.css';

const Home = () => {
  return (
    <div className={styles.container}>
      <h1>ホームページ</h1>
      {/* コンテンツを追加 */}
    </div>
  );
};

export default Home;
```

### よくあるエラー
1. **モジュールが見つからない**: `npm install`を実行
2. **ポートが使用中**: 別のポート(5174)を使用
3. **CSS Modulesが効かない**: ファイル名が`.module.css`になっているか確認
